# <a name="configure-dsc-lcm-with-new-meta-configuration-attribute"></a>Konfigurera DSC MGM med nya metadata-konfiguration-attribut

Den `DscLocalConfigurationManager` attributet betecknar ett block med konfigurationen som en meta-konfiguration, som används för att konfigurera DSC Local Configuration Manager.

Mer information finns i [konfigurerar den lokala Konfigurationshanteraren](https://msdn.microsoft.com/powershell/dsc/metaconfig).
