# <a name="specifying-cross-node-dependencies"></a>Ange beroenden mellan noder

Med hjälp av inbyggda WaitFor\* resurser (`WaitForAll`, `WaitForAny`, och `WaitForSome`), kan du nu ange beroenden mellan datorer under configuration körs utan externa orchestration.

Mer information finns i [angett beroenden mellan noder](https://msdn.microsoft.com/powershell/dsc/crossnodedependencies)
