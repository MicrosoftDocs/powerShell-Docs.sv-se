# <a name="direct-access-to-dsc-resource-methods"></a>Direkt åtkomst till metoder för DSC-resurs


Den [Invoke-DscResource](https://technet.microsoft.com/en-us/library/mt517869.aspx) cmdlet har lagts till direkt åtkomst till DSC-resurser och deras metoder (Get, Set eller Test). Den kan användas av andra företag som vill dra nytta av DSC-resurser.

Mer information finns i [anropar DSC resursen direkt metoder](https://msdn.microsoft.com/powershell/dsc/directcallresource).
