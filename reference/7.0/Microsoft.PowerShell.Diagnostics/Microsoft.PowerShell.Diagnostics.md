# Microsoft. PowerShell. Diagnostics-modul

## Beskrivning

Det här avsnittet innehåller hjälp avsnitten för de cmdletar som är installerade med PowerShell-modulen Microsoft. PowerShell. Diagnostics, som innehåller cmdletar som hanterar data från händelse loggar.

## Microsoft. PowerShell. Diagnostics-cmdletar

### [Hämta räknare](Get-Counter.md)
Hämtar prestanda räknar data från lokala datorer och fjärrdatorer.

### [Get-WinEvent](Get-WinEvent.md)
Hämtar händelser från händelse loggar och händelse spårnings loggar på lokala datorer och fjärrdatorer.

### [New-WinEvent](New-WinEvent.md)
Skapar en ny Windows-händelse för den angivna händelse leverantören.
