# Microsoft. PowerShell. Core-modul

## Beskrivning

Det här avsnittet innehåller hjälp avsnitten för de cmdletar som är installerade med PowerShell-modulen Microsoft. PowerShell. Core. Huvud-modulen innehåller cmdlets och providers som hanterar de grundläggande funktionerna i PowerShell.
