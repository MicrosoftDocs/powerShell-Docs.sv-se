# Clear-ItemProperty

## SAMMANFATTNING
Rensar värdet för en egenskap men tar inte bort egenskapen.

## SYNTAX

### Sökväg (standard)

```
Clear-ItemProperty [-Path] <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>] [-WhatIf] [-Confirm]
 [-UseTransaction] [<CommonParameters>]
```

### LiteralPath

```
Clear-ItemProperty -LiteralPath <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>] [-WhatIf] [-Confirm]
 [-UseTransaction] [<CommonParameters>]
```

## BESKRIVNING

`Clear-ItemProperty`Cmdleten rensar värdet för en egenskap, men den tar inte bort egenskapen.
Du kan använda den här cmdleten för att ta bort data från ett register värde.

## EXEMPEL

### Exempel 1: Rensa värdet för register nyckeln

Det här kommandot rensar data i registervärdet "alternativ" i under nyckeln "mitt" i `HKEY_LOCAL_MACHINE\Software\MyCompany` .

```powershell
Clear-ItemProperty -Path "HKLM:\Software\MyCompany\MyApp" -Name "Options"
```

## ANTECKNINGAR

Du kan använda `Clear-ItemProperty` för att ta bort data i register värden utan att ta bort värdet. Om värdets datatyp är Binary eller DWORD, ställer du in värdet på noll om du rensar data. Annars är värdet tomt.

`Clear-ItemProperty`Cmdleten är utformad för att fungera med data som exponeras av vilken provider som helst. Om du vill visa en lista över tillgängliga providers i din session skriver du `Get-PSProvider` . Mer information finns i about_Providers.
