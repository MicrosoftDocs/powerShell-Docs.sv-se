# Microsoft. PowerShell. Host-modul

## Beskrivning

Det här avsnittet innehåller hjälp avsnitten för de cmdletar som är installerade med PowerShell-modulen Microsoft. PowerShell. Host. Modulen värd innehåller cmdletar som hanterar data från värd program.

## Microsoft. PowerShell. värd-cmdletar

### [Start-avskrift](Start-Transcript.md)
Skapar en post med hela eller delar av en PowerShell-session till en textfil.

### [Stopp-avskrift](Stop-Transcript.md)
Stoppar en avskrift.
