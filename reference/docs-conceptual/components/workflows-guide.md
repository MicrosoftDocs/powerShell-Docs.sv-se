# <a name="psworkflow-module"></a>PSWorkflow-modul

Det här avsnittet innehåller hjälp avsnitten för **PSWorkflow** -modulen, som innehåller cmdletar som har stöd för Windows PowerShell Workflow-funktionen. **PSWorkflow** -modulen installeras med Windows PowerShell 5,1.

> [!NOTE]
> Arbets flöden stöds bara i Windows PowerShell 5,1.
