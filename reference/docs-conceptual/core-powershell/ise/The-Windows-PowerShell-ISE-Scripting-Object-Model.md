# <a name="the-windows-powershell-ise-scripting-object-model"></a>Windows PowerShell ISE-skriptobjektmodellen
  Windows PowerShell® Integrated Scripting Environment (ISE) visar dess funktioner med hjälp av en underliggande skript objektmodell som gör det möjligt för användare att skriva Windows PowerShell-skript som manipulera olika grafiska och funktionella aspekter av den Windows PowerShell ISE. Skriptobjekt, deras egenskaper och deras metoder definieras specifikt för detta ändamål.
