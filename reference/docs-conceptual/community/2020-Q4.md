## <a name="2020-november-highlights"></a>2020 – november-höjder

- PowerShell 7,1 GA-version
  - Nyheter i PowerShell 7,1
  - Konverterade 7,1-dokument till versions status
  - Ytterligare 7,2-dokument (för hands version)
  - Indragna V6-dokument till Arkiv lag rings platsen

- Blogginlägg
  - [Du har hjälp!](https://devblogs.microsoft.com/powershell/youve-got-help/)
  - [Uppdatera hjälp för PSReadLine-modulen](https://devblogs.microsoft.com/powershell/updating-help-for-the-psreadline-module/)

- Dokumentations underhåll
  - Uppdaterade 137-artiklar för att ta bort MSDN-och TechNet-referenser
  - Uppdaterade 171-artiklar för att visa Windows-kompatibilitet
  - Uppdaterade 38-artiklar för att hantera build-varningar och förslag
  - Ingår i 24 DSC-artiklar
  - Större omskrivning av artiklar med PowerShell-jobb
    - [about_Jobs](/powershell/module/microsoft.powershell.core/about/about_jobs)
    - [about_Remote_Jobs](/powershell/module/microsoft.powershell.core/about/about_remote_jobs)
    - [about_Thread_Jobs](/powershell/module/microsoft.powershell.core/about/about_thread_jobs)

- GitHub statistik
  - 50 pull slogs samman (8 från community)
  - 55 problem öppnade (45 från communityn)
  - 51 problem har stängts
