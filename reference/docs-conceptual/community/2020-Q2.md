# <a name="whats-new-in-powershell-docs"></a>Nyheter i PowerShell Docs

Den här artikeln innehåller några av de större ändringarna i dokumenten under föregående månad och firar bidragen från communityn.

Hjälp oss att göra dokumentationen bättre. Läs [bidrags hand boken][contrib] för att lära dig hur du kommer igång.

## <a name="2020-may-highlights"></a>2020 – maj-högdagrar

- Ny PowerShell-dokumentation
  - Skapat ett nytt [djup dykningar](../learn/deep-dives/overview.md) -avsnitt som innehåller innehåll från community Contributor Kevin Marquette
    - [Allt du vill veta om matriser](../learn/deep-dives/everything-about-arrays.md)
    - [Allt du vill veta mer om hashtabeller](../learn/deep-dives/everything-about-hashtable.md)
    - [Allt du vill veta mer om PSCustomObject](../learn/deep-dives/everything-about-pscustomobject.md)
    - [Allt du vill veta om strängersättning](../learn/deep-dives/everything-about-string-substitutions.md)
    - [Allt du vill veta om if/then/else](../learn/deep-dives/everything-about-if.md)
    - [Allt du vill veta om byte](../learn/deep-dives/everything-about-switch.md)
    - [Allt du vill veta om undantag](../learn/deep-dives/everything-about-exceptions.md)
    - [Allt du vill veta om $null](../learn/deep-dives/everything-about-null.md)
    - [Allt du vill veta om ShouldProcess](../learn/deep-dives/everything-about-shouldprocess.md)
    - [Såhär skapar du en binär modul för standardbibliotek](../dev-cross-plat/create-standard-library-binary-module.md)
  - Publicerade [PowerShell 7,0 .NET API](/dotnet/api/?view=powershellsdk-7.0.0) -referensen
  - `Update-Help -Force`för PowerShell 5,1 hämtar nu uppdaterat innehåll för de viktigaste PowerShell-modulerna
- Dokumentations underhåll
  - Större organisation av innehålls förteckningen
    - Nytt innehåll under **Learning PowerShell**
    - Windows PowerShell 5,1-innehåll samlas in på en plats
  - Arkiverat äldre innehåll till[https://aka.ms/PSLegacyDocs](https://aka.ms/PSLegacyDocs)
    - PowerShell-Webbåtkomst
    - Guide för PowerShell-arbetsflöden
  - Uppdateringar för 7,1 – för hands versions innehåll

- GitHub statistik
  - 81 pull slogs samman (21 från community)
  - 61 problem öppnade (53 från communityn)
  - 64 problem har stängts
