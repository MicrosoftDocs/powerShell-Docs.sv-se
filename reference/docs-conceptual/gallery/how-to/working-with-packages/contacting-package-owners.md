# <a name="contacting-package-owners"></a>Kontaktar paket ägare

Om du vill kontakta ägaren av ett visst paket går du till sidan paket information. Det finns en länk till kontakt ägare i den vänstra meny raden. Om du klickar på länken tas du till en ny sida. Du kan skicka ett meddelande härifrån.
