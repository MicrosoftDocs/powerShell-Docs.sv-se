# <a name="cmdlet-development-guidelines"></a>Riktlinjer för cmdlet-utveckling

Ämnena i det här avsnittet innehåller utvecklings rikt linjer som du kan använda för att skapa välformulerade cmdletar. Genom att använda de vanliga funktionerna i Windows PowerShell-körningsmiljön och genom att följa dessa rikt linjer kan du utveckla robusta cmdlets med minimal ansträngning och ge användaren en konsekvent upplevelse. Dessutom kommer du att minska test belastningen eftersom vanliga funktioner inte kräver omtestering.
