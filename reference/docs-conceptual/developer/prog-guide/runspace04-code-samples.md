# <a name="runspace04-code-samples"></a>RunSpace04 – kodexempel

Här är ett kod exempel för en körnings utrymme som använder klassen [system. Management. Automation. Runspaceinvoke](/dotnet/api/System.Management.Automation.RunspaceInvoke) för att köra ett skript som genererar ett avslutande fel. Värd programmet ansvarar för att fånga upp felet och tolka fel posten.

> [!NOTE]
> Du kan ladda ned VB.NET-källfilen (Runspace04. VB) för den här körnings utrymme med Windows Software Development Kit för Windows Vista och Microsoft .NET Framework 3,0 Runtime-komponenter. Instruktioner för hämtning finns i [Installera Windows PowerShell och ladda ned Windows POWERSHELL SDK](/powershell/scripting/developer/installing-the-windows-powershell-sdk).
>
> De hämtade källfilerna är tillgängliga i **\<PowerShell Samples>** katalogen.
