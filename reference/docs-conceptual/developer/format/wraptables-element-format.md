# <a name="wraptables-element-format"></a>WrapTables-element (format)

Anger att data i en tabell cell flyttas till nästa rad om data är längre än kolumnens bredd.

Konfigurations element (format) DefaultSettings-element (format) WrapTables-element (format)

## <a name="syntax"></a>Syntax

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Attribut och element

I följande avsnitt beskrivs attribut, underordnade element och `WrapTables` elementets överordnade element.

### <a name="attributes"></a>Attribut

Inga.

### <a name="child-elements"></a>Underordnade element

Inga.
