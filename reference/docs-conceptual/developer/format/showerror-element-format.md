# <a name="showerror-element-format"></a>ShowError-element (format)

Anger att den fullständiga fel posten visas när ett fel inträffar när en del av data visas.

Konfigurations element (format) DefaultSettings-element (format) ShowError-element (format)

## <a name="syntax"></a>Syntax

```scr
<ShowError/>
```

## <a name="attributes-and-elements"></a>Attribut och element

I följande avsnitt beskrivs attribut, underordnade element och det överordnade elementet i `ShowError`-elementet. Standardvärdet för det här elementet är `false`.

### <a name="attributes"></a>Attribut

Ingen.

### <a name="child-elements"></a>Underordnade element

Ingen.
