# <a name="enumeratecollection-element-for-expressionbinding-for-controls-for-configuration-format"></a>EnumerateCollection-element för ExpressionBinding för Controls för Configuration (format)

Anger att elementen i samlingar visas av kontrollen. Det här elementet används när du definierar en gemensam kontroll som kan användas av alla vyer i format filen.

Konfigurations element (format) styr element i konfigurations-(format) kontroll element för Controls (format) CustomControl-element för Control för Configuration (format) CustomEntries-element för CustomControl för konfiguration ( Format) CustomEntry-element för CustomControl for Controls for Configuration (format) CustomItem element for CustomEntry for Controls for Configuration ExpressionBinding element for CustomItem for Controls for Configuration (format) EnumerateCollection Element för ExpressionBinding för konfigurations kontroller (format)
