# <a name="newline-element-for-customitem-for-controls-for-configuration-format"></a>NewLine-element för CustomItem för Controls för Configuration (format)

Lägger till en tom rad i visningen av kontrollen. Det här elementet används när du definierar en gemensam kontroll som kan användas av alla vyer i format filen.

Konfigurations element (format) styr element i konfigurations-(format)-kontroll element för Controls (format) CustomControl-element för Control för Configuration (format) CustomEntries-element för CustomControl för Configuration (format) CustomEntry-element för CustomControl för Controls (format) CustomItem-element för CustomEntry för kontroller för konfigurations element för konfiguration (format)

## <a name="syntax"></a>Syntax

```xml
<NewLine/>
```
