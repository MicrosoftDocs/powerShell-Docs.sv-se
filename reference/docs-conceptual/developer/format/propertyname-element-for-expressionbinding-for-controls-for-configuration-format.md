# <a name="propertyname-element-for-expressionbinding-for-controls-for-configuration-format"></a>PropertyName-element för ExpressionBinding för Controls för Configuration (format)

Anger .NET-egenskapen vars värde visas av den gemensamma kontrollen. Det här elementet används när du definierar en gemensam kontroll som kan användas av alla vyer i format filen.
