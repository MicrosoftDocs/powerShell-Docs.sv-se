# <a name="whats-new-with-powershell-50"></a>Vad är nytt i PowerShell 5.0
Windows PowerShell® 5.0 innehåller nya viktiga funktioner som utökar användningen, förbättrar dess användbarhet och gör att du kan styra och hantera Windows-baserade miljöer enklare och mer omfattande.  Stora förbättringar är inom områden med Desired State Configuration, säkerhet, prestanda, fjärrkommunikation och förbättringar av språk.

Windows PowerShell 5.0 är bakåtkompatibla. Cmdlet: ar, leverantörer, moduler, snapin-moduler, skript, funktioner och profiler som har utformats för Windows PowerShell 4.0, Windows PowerShell 3.0 och Windows PowerShell 2.0 Allmänt fungerar i Windows PowerShell 5.0 utan ändringar.

Windows PowerShell 5.0 installeras som standard på Windows Server® 2016 och Windows® 10. Om du vill installera Windows PowerShell 5.0 på Windows Server 2012 R2, Windows 8.1 Enterprise eller Windows 8.1 Pro, ladda ned och installera [Windows Management Framework 5.0](https://go.microsoft.com/fwlink/?linkid=830436). Glöm inte att läsa hämta information och uppfyller alla systemkrav, innan du installerar Windows Management Framework 5.0.
