# <a name="whats-new-with-powershell-50"></a>Vad är nytt med PowerShell 5,0
Windows PowerShell® 5,0 innehåller viktiga nya funktioner som utökar användningen, förbättrar dess användbarhet och gör att du kan styra och hantera Windows-baserade miljöer enklare och mer omfattande.  De större förbättringarna är i områden med önskad tillstånds konfiguration, säkerhet, prestanda, fjärr kommunikation och språk förbättringar.

Windows PowerShell 5,0 är bakåtkompatibelt. Cmdlets, providers, moduler, snapin-moduler, skript, funktioner och profiler som har utformats för Windows PowerShell 4,0, Windows PowerShell 3,0 och Windows PowerShell 2,0 fungerar vanligt vis i Windows PowerShell 5,0 utan ändringar.

Windows PowerShell 5,0 installeras som standard på Windows Server® 2016 och Windows® 10. Installera Windows PowerShell 5,0 på Windows Server 2012 R2, Windows 8,1 Enterprise eller Windows 8,1 Pro genom att hämta och installera [Windows Management Framework 5,0](https://go.microsoft.com/fwlink/?linkid=830436). Se till att läsa informationen om hämtningen och uppfyller alla system krav innan du installerar Windows Management Framework 5,0.
