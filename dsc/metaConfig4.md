### <a name="example-of-updating-local-configuration-manager-settings"></a><span data-ttu-id="c1a05-147">Exempel för att uppdatera inställningarna för Local Configuration Manager</span><span class="sxs-lookup"><span data-stu-id="c1a05-147">Example of updating Local Configuration Manager settings</span></span>

<span data-ttu-id="c1a05-148">Du kan uppdatera Local Configuration Manager-inställningarna för en målnod genom att inkludera en **LocalConfigurationManager** blockera i nod-block i ett konfigurationsskript som visas i följande exempel.</span><span class="sxs-lookup"><span data-stu-id="c1a05-148">You can update the Local Configuration Manager settings of a target node by including a **LocalConfigurationManager** block inside the node block in a configuration script, as shown in the following example.</span></span>

```powershell
Configuration ExampleConfig
{
    Node “Server001”
    {
        LocalConfigurationManager
        {
            ConfigurationID = "646e48cb-3082-4a12-9fd9-f71b9a562d4e"
            ConfigurationModeFrequencyMins = 45
            ConfigurationMode = "ApplyAndAutocorrect"
            RefreshMode = "Pull"
            RefreshFrequencyMins = 90
            DownloadManagerName = "WebDownloadManager"
            DownloadManagerCustomData = (@{ServerUrl="https://$PullService/psdscpullserver.svc"})
            CertificateID = "71AA68562316FE3F73536F1096B85D66289ED60E"
            Credential = $cred
            RebootNodeIfNeeded = $true
            AllowModuleOverwrite = $false
        }
# One or more resource blocks can be added here
    }
}

# The following line invokes the configuration and creates a file called Server001.meta.mof at the specified path
ExampleConfig -OutputPath "c:\users\public\dsc"
```

<span data-ttu-id="c1a05-149">Kör skriptet i föregående exempel genererar en MOF-fil som anger och lagrar de önskade inställningarna.</span><span class="sxs-lookup"><span data-stu-id="c1a05-149">Running the script in the previous example generates a MOF file that specifies and stores the desired settings.</span></span>
<span data-ttu-id="c1a05-150">Om du vill använda inställningarna du kan använda den **Set DscLocalConfigurationManager** cmdlet, som visas i följande exempel.</span><span class="sxs-lookup"><span data-stu-id="c1a05-150">To apply the settings, you can use the **Set-DscLocalConfigurationManager** cmdlet, as shown in the following example.</span></span>

```powershell

```
