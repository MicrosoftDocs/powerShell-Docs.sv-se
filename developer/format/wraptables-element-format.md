# <a name="wraptables-element-format"></a>WrapTables-element (format)

Anger att data i en tabellcell flyttas till nästa rad om data är längre än bredden på kolumnen.

Elementet (Format) DefaultSettings Element (Format) WrapTables konfigurationselement (Format)

## <a name="syntax"></a>Syntax

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Attribut och element

I följande avsnitt beskrivs attribut, underordnade element och det överordnade elementet i den `WrapTables` element.

### <a name="attributes"></a>Attribut

Ingen.

### <a name="child-elements"></a>Underordnade element

Ingen.

### <a name="parent-elements"></a>Överordnade element

|Element|Beskrivning|
|-------------|-----------------|
|[DefaultSettings Element (Format)](./defaultsettings-element-format.md)|Definierar gemensamma inställningar som gäller för alla vyer i filen formatering.|
