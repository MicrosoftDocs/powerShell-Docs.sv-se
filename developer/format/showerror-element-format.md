# <a name="showerror-element-format"></a>ShowError-element (format)

Anger att en fullständig Felpost visas när ett fel uppstår medan du visar en typ av data.

Elementet (Format) DefaultSettings Element (Format) ShowError konfigurationselement (Format)

## <a name="syntax"></a>Syntax

```scr
<ShowError/>
```

## <a name="attributes-and-elements"></a>Attribut och element

I följande avsnitt beskrivs attribut, underordnade element och det överordnade elementet i den `ShowError` element. Standardvärdet för det här elementet är `false`.

### <a name="attributes"></a>Attribut

Ingen.

### <a name="child-elements"></a>Underordnade element

Ingen.
