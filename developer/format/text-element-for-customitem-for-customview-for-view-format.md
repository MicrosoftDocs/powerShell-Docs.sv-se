# <a name="text-element-for-customitem-for-customview-for-view-format"></a>Text-element för CustomItem för CustomView för View (format)

Anger text som läggs till i de data som visas av kontroll, till exempel en etikett, hakparenteser för att skriva data och blanksteg dra in data. Det här elementet används när du definierar en anpassad kontroll vy.

Elementet (Format) ViewDefinitions Element (Format) visa Element (Format) anpassad kontroll Element (Format) CustomEntries konfigurationselement för anpassad kontroll för Visa (Format) CustomEntry elementet för CustomEntries för vyn (Format) CustomItem elementet för CustomEntry för Visa (Format) textelement för CustomItem (Format)

## <a name="syntax"></a>Syntax

```xml
<Text>TextToDisplay</Text>
```

## <a name="attributes-and-elements"></a>Attribut och element

I följande avsnitt beskrivs attribut, underordnade element och det överordnade elementet i den `Text` element.

### <a name="attributes"></a>Attribut

Ingen.

### <a name="child-elements"></a>Underordnade element

Ingen.

|Element|Beskrivning|
|-------------|-----------------|

## <a name="text-value"></a>Textvärde

Ange text i en kontroll för data som du vill visa.
