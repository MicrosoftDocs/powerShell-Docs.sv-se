# <a name="windows-powershell-error-reporting"></a>Windows PowerShell-felrapportering

Ämnena i det här avsnittet beskrivs hur cmdletar rapporterar fel.

## <a name="in-this-section"></a>I detta avsnitt

[Fel Reporting begrepp](./error-reporting-concepts.md) beskriver de två mekanismer som cmdlet: ar kan använda för att rapportera fel.

[Avslutande fel](./terminating-errors.md) beskriver den metod som används för att rapportera avslutande fel, där den metoden kan anropas från cmdlet: en, och undantag som kan returneras av Windows PowerShell-körningen när metoden anropas.

[Icke-avslutande fel](./non-terminating-errors.md) beskriver den metod som används för att rapportera icke-avslutande fel och där den metoden kan anropas från i cmdleten.

[Visa Information om fel efter kategori](./displaying-error-information.md) beskrivs de metoder för att användare kan visa fel.

[Windows PowerShell-felposter](./windows-powershell-error-records.md) beskriver komponenterna i en Felpost.

[Tolka ErrorRecord objekt](./interpreting-errorrecord-objects.md) diskuterar hur ErrorRecord objekt ska tolkas.
