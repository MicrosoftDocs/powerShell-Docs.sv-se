# <a name="remote-runspace-samples"></a>Fjärranslutna körningsutrymmen – exempel

Det här avsnittet innehåller exempelkod som visar hur du skapar körningsutrymmen som kan användas för att ansluta till en dator med hjälp av WS-Management-baserade Windows PowerShell-fjärrkommunikation. Du kan använda Microsoft Visual Studio för att skapa ett konsolprogram och sedan kopiera koden från avsnitten i det här avsnittet i din värdapp.

## <a name="in-this-section"></a>I detta avsnitt

> [!NOTE]
> Läs mer om hur du kör kommandon på en fjärrdator, [Windows PowerShell-fjärrkommunikation](https://msdn.microsoft.com/en-us/library/ee706563(v=vs.85).aspx).

 [RemoteRunspace01 exempel](./remoterunspace01-sample.md) i det här exemplet visar hur du skapar ett fjärrkörningsutrymme som används för att upprätta en fjärranslutning.

 [RemoteRunspacePool01 exempel](./remoterunspacepool01-sample.md) det här exemplet visas hur du skapar poolen fjärrkörningsutrymme och hur du kör flera kommandon samtidigt med hjälp av den här poolen.
